= Désinstaller le courtier de données pour NetApp Copy and Sync
:hardbreaks:
:allow-uri-read: 
:nofooter: 
:icons: font
:linkattrs: 
:imagesdir: ./media/


[role="lead"]
Si nécessaire, exécutez un script de désinstallation pour supprimer le courtier de données ainsi que les packages et répertoires créés pour NetApp Copy and Sync lors de l'installation du courtier de données.

.Étapes
. Connectez-vous à l'hôte du courtier de données.
. Passer au répertoire du courtier en données : `/opt/netapp/databroker`
. Exécutez les commandes suivantes :
+
`chmod +x uninstaller-DataBroker.sh`
`./uninstaller-DataBroker.sh`

. Appuyez sur « y » pour confirmer la désinstallation.
